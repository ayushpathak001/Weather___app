# 🌤 Weather App

A modern and responsive Weather App that provides real-time weather updates with location autocomplete and a beautiful glassmorphism UI.

---

## 🚀 Features

* 🔍 Search weather by city name
* 📍 Location autocomplete (accurate results)
* 🌡 Real-time temperature (°C / °F toggle)
* ☁ Weather condition with icon
* 💧 Humidity & 💨 Wind speed
* ⏳ Loading state & error handling
* 🎨 Dynamic background based on weather
* 📱 Fully responsive design
* ⌨ Press **Enter** to search

---

## 🛠 Tech Stack

* HTML5
* CSS3 (Glassmorphism UI)
* JavaScript (ES6)
* WeatherAPI
